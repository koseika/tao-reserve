# Tao Reserve サイト構成

このファイルは、サイト全体の構成と役割を整理するためのメモです。\
README.md
はシステム仕様・バージョン管理用、このファイルはサイト設計用として管理します。

------------------------------------------------------------------------

## サイト構成

``` text
index.html
└── トップページ

worldview.html
└── タオタントラの世界観

session.html
└── セッションについて

price.html
└── 料金・ご予約

calendar.html
└── 空き情報・予約

admin.html
└── 予約管理画面（一般公開しない）

style.css
└── 共通デザイン

script.js
└── 予約カレンダー表示処理

reservation-data.js
└── 予約データ処理

firebase-config.js
└── Firebase接続設定
```

------------------------------------------------------------------------

## データの流れ

``` text
admin.html
      │
      ▼
 Firebase
      ▲
      │
calendar.html
```

管理画面（admin.html）から予約状況を変更すると、
Firebaseへ保存され、その内容を予約ページ（calendar.html）が読み込んで表示します。

HTMLを書き換えているわけではありません。

------------------------------------------------------------------------

## デザイン方針

-   トップページは一画面で完結させる
-   できるだけスクロールしない
-   世界観を説明しすぎない
-   静かな入口を意識する
-   必要な人が、必要なページへ自然に進める構成にする
